== About Extensions
The {product} {product-very-short} instance includes the Extensions plugin preinstalled and enabled by default. Your organization can use the Extensions plugin to discover plugins which are modular extensions for {product-very-short} that extend functionality, streamline development workflows, and improve the developer experience.
